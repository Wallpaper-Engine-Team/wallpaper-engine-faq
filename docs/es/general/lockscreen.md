# Wallpaper Engine en la pantalla de bloqueo de Windows

Nos suelen llegar preguntas sobre si es posible configurar un fondo de Wallpaper Engine en la pantalla de bloqueo de Windows. Por desgracia, no es posible debido a las limitaciones del propio Windows. La pantalla de bloqueo es una zona de Windows protegida que no se puede modificar sin hackear Windows, This serves system security as applications should not be able to add malicious content to the login window. For this reason, Wallpaper Engine cannot add wallpapers to the lock screen and we cannot add this feature in the future either.
