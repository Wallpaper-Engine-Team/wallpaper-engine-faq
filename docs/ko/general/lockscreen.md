# Windows 잠금 화면에서 Wallpaper Engine

종종 Windows 잠금 화면에서 Wallpaper Engine 배경화면을 설정할 수 있는지 질문을 받습니다. 안타깝게도 Windows 자체의 제한으로 불가능합니다. The lock screen is a protected area of Windows which cannot be modified without hacking Windows. This serves system security as applications should not be able to add malicious content to the login window. For this reason, Wallpaper Engine cannot add wallpapers to the lock screen and we cannot add this feature in the future either.
